# React Portfolio – JKR

## Overview  
This is a modern, animation-driven developer portfolio built using React, Vite, TailwindCSS, and a collection of visual/interaction libraries. It highlights skills, experience, and key projects through a clean interface with smooth motion effects. The site is fully deployed via GitHub Pages.

## Live Demo  
**Portfolio:**  
https://Jamal-Raja.github.io/react-portfolio-jkr/

---

## Features

### Home  
- Animated hero section with LiquidEther background.  
- Typewriter-style introduction text.  
- Chevron link for smooth in-page navigation.  
- Quick link to the Test page.

### About  
- AOS-powered scroll-triggered animations.  
- Brief intro, experience summary, and future goals.  
- Categorised skill groups covering both Front-end and Back-end stacks.

### Projects  
- Particle-based animated background.  
- Interactive TiltedCard hover effects.  
- Each project includes:
  - Title  
  - Screenshot  
  - Technology stack  
  - Description  
  - GitHub repo link  
  - Live app link  

---

## Technology Stack

### Core  
- React 19  
- Vite  
- React Router DOM 7  
- TailwindCSS 4  

### Animation & Visuals  
- AOS  
- GSAP  
- Motion  
- OGL  
- Three.js  
- Custom components: LiquidEther, Particles, TextType, DecryptedText, TiltedCard
